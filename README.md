# Computation-and-Visualization-of-Data
 Coursework from **IE6600 – Computation and Visualization of Data** at Northeastern   University (Spring 2026).    Covers the full Python visualization stack — from foundational Matplotlib to   interactive dashboards   with Plotly and Altair, statistical visualization with Seaborn, image processing with    OpenCV,   and visualization critique.
